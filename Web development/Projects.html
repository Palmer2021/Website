Computer systems project

For the coursework in this module we had to create a circuit using digital works that using logic gates we could use a number display to show the year and your student ID number. We had to colour it and also do calculations showing how we worked out what inputs we would need to make the circuit work.We did this by getting an equation which in turn could be used to decide what inputs to do.

Introduction to programming project

In this project we had a scenario about train seat prices and bookings. We had to read a file of 4 seats and their prices with a fixed discount rate. When you run the program it asks if you want to specify a different discount rate or use the discount one. If the user wants to user a new discount rate then it calculates the new discounts. It also calculates and displays the total discounts and income with the new discounts.
